# Privacy Policy

**Last updated:** May 5th 2026

## 1. Overview

This repository provides source code only. It does **not** provide a hosted application or service.

The developer of this project does **not collect, receive, store, or process any personal or health data** from users.

---

## 2. No Data Collection by Developer

* The developer does **not operate any servers or backend services** for this project
* The developer does **not have access to any user data**, including data from the Oura API
* The developer does **not receive OAuth tokens, credentials, or health data**

All data processing happens **locally or within infrastructure controlled by the user running the software**.

## 3. User-Controlled Data

If you choose to use this code:

* You are solely responsible for connecting your own accounts (e.g., Oura)
* Any data retrieved from third-party services is handled entirely within your own environment
* You are responsible for how your data is stored, secured, and processed

## 4. Third-Party Services

This project may interact with third-party APIs such as the Oura API.

* Any data access is performed directly between your environment and the third-party service
* The developer is **not involved in this data exchange**

## 5. Data Storage

## 6. Data Sharing
